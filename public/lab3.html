Assembly Language


  1. Write a program that will prompt the user to enter
    a hex digit character (“0” … “9” or “A” … “F”), display it on the next line in decimal, and ask the user if he or she wants to do it again. If the user types “y” or “Y”,
    the program repeat; if the user types anything else,
    the program terminates. If the user enters an illegal character, prompt the user to try again.    (LAB31.ASM)

Sample execution:

ENTER A HEX DIGIT: 9

IN DECIMAL IT IS 9

DO YOU WANT TO DO IT AGAIN? Y

ENTER A HEX DIGIT: c

ILLEGAL CHARACTER – ENTER 0..9 OR A..F: C

IN DECIMAL IT IS 12

DO YOU WANT TO DO IT AGAIN? N
